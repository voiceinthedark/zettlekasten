## array Operations

### initialization
```php
$myarray = [1, 3, 2.3, 'ss', null];
$myMap = [
    'One' => 1,
    'Two' => 2,
    'Three' => 3
];
```

### printing arrays
```php
var_dump($array);
print_r($array);

// readable arrays
echo '<pre>';
print_r($array);
echo '</pre>';
```

### check if array element exists
```php
isset($array[2]);
```

### Length of an array
```php
count($array);
```

### Add element to an array
```php
$array[] = 6;
array_push($array, 6);
```

### remove element from an array
```php
array_pop($array); // removes last element
array_shift($array); // removes first element
```

### check if key exists
```php
array_key_exists('a', $array);
```
